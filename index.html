Windows hardening scripts for STIG compliance and security automation.

WN11-CC-000305

Disable indexing of encrypted files to prevent data exposure.

View Script 
# Disable indexing of encrypted files
$Path = "HKLM:\SOFTWARE\Policies\Microsoft\Windows\Windows Search"

if (-not (Test-Path $Path)) {
    New-Item -Path $Path -Force | Out-Null
}

New-ItemProperty -Path $Path `
    -Name "AllowIndexingEncryptedStoresOrItems" `
    -PropertyType DWord `
    -Value 0 `
    -Force | Out-Null

WN11-00-000160

Disable SMBv1 (legacy insecure protocol).

View Script 
Disable-WindowsOptionalFeature -Online -FeatureName SMB1Protocol -NoRestart

Set-ItemProperty -Path "HKLM:\SYSTEM\CurrentControlSet\Services\LanmanServer\Parameters" `
    -Name SMB1 -Value 0 -Type DWord
