PowerShell STIG Implementations

Collection of STIG remediation scripts used for Windows hardening and cybersecurity compliance automation.

STIG WN11-CC-000305

Enforces Windows Defender real-time protection.

Set-MpPreference -DisableRealtimeMonitoring $false

Ensures endpoint protection is always active to prevent malware execution.

STIG WN11-CC-000252

Configures logon auditing.

auditpol /set /subcategory:"Logon" /success:enable /failure:enable

Enables tracking of successful and failed logon attempts.

STIG WN11-CC-000326

Password policy enforcement.

net accounts /minpwlen:14
net accounts /complexity:enabled

Strengthens authentication security against brute-force attacks.

STIG WN11-CC-000160

Disables SMBv1 protocol.

Disable-WindowsOptionalFeature -Online -FeatureName SMB1Protocol -NoRestart

Removes vulnerable legacy file-sharing protocol.

STIG WN11-CC-000005

Enables UAC (User Account Control).

Set-ItemProperty -Path "HKLM:\Software\Microsoft\Windows\CurrentVersion\Policies\System" `
-Name "EnableLUA" -Value 1

Prevents silent privilege escalation.

STIG WN11-CC-000345

Screen lock timeout enforcement.

powercfg /change monitor-timeout-ac 15

Locks idle systems to reduce unauthorized access risk.

STIG WN11-CC-000285

Disables Guest account.

net user guest /active:no

Prevents unauthorized guest logins.

STIG WN11-CC-000090

Enables object access auditing.

auditpol /set /category:"Object Access" /success:enable /failure:enable

Tracks file and registry access for forensic investigations.

STIG WN11-CC-000120

Enables Windows Firewall (Domain Profile).

Set-NetFirewallProfile -Profile Domain -Enabled True

Ensures network filtering is enforced on domain networks.

STIG WN11-CC-000200

Disables AutoRun for removable media.

Set-ItemProperty -Path "HKLM:\Software\Microsoft\Windows\CurrentVersion\Policies\Explorer" `
-Name "NoDriveTypeAutoRun" -Value 255

Prevents malware execution from USB devices.
