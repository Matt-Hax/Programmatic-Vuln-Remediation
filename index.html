PowerShell STIG Implementations

This repository contains PowerShell scripts used to remediate Windows STIG controls and enforce security compliance.

1. STIG WN11-CC-000305

Description: Enforces security settings related to Windows Defender behavior.

# Example Script
Set-MpPreference -DisableRealtimeMonitoring $false

Explanation: This ensures real-time protection is enabled to prevent malware execution.

2. STIG WN11-CC-000252

Description: Configures audit policy settings for logon events.

auditpol /set /subcategory:"Logon" /success:enable /failure:enable

Explanation: This ensures successful and failed logon attempts are logged for investigation.

3. STIG WN11-CC-000326

Description: Controls password complexity enforcement.

net accounts /minpwlen:14
net accounts /complexity:enabled

Explanation: Forces strong password policies to reduce brute-force risk.

4. STIG WN11-CC-000160

Description: Disables SMBv1 protocol.

Disable-WindowsOptionalFeature -Online -FeatureName SMB1Protocol -NoRestart

Explanation: Removes legacy SMBv1 to prevent exploitation like EternalBlue.

5. STIG WN11-CC-000005

Description: Configures UAC behavior.

Set-ItemProperty -Path "HKLM:\Software\Microsoft\Windows\CurrentVersion\Policies\System" `
-Name "EnableLUA" -Value 1

Explanation: Ensures User Account Control is enabled to restrict privilege escalation.

6. STIG WN11-CC-000345

Description: Enforces screen lock timeout.

powercfg /change monitor-timeout-ac 15

Explanation: Locks inactive systems quickly to prevent unauthorized access.

7. STIG WN11-CC-000285

Description: Disables guest account.

net user guest /active:no

Explanation: Prevents unauthorized guest access to the system.

8. STIG WN11-CC-000090

Description: Enables audit object access.

auditpol /set /category:"Object Access" /success:enable /failure:enable

Explanation: Tracks file and registry access for forensic analysis.

9. STIG WN11-CC-000120

Description: Enforces firewall domain profile.

Set-NetFirewallProfile -Profile Domain -Enabled True

Explanation: Ensures firewall is active on domain networks.

10. STIG WN11-CC-000200

Description: Disables auto-run for removable media.

Set-ItemProperty -Path "HKLM:\Software\Microsoft\Windows\CurrentVersion\Policies\Explorer" `
-Name "NoDriveTypeAutoRun" -Value 255

Explanation: Prevents malware execution from USB devices.
